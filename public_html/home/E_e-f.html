Nome: Mercado (Market)
Custo: 175
Pontos de vida: 2100
Ataque: 0
Guarnição: 0
Alcance: 0

 

Descrição: O Mercado permite negociar com outros jogadores por terra, comprar e vender recursos e oferecer recursos a outros jogadores como tributo. É usado também para pesquisar tecnologias que aprimoram sua comunicação com os aliados e diminiu o custo do comércio de mercadorias e tributos.
Você deve ter um Moinho para poder construir um Mercado.

Nome: Armadilha para Peixes (Fish Trap)
Custo: 100
Pontos de vida: 50
Ataque: 0
Guarnição: 0
Alcance: 0

 

Descrição: As Armadilhas para Peixes fornecem uma renovável fonte de alimentos. Elas são construídas por um navio pesqueiro, portanto você deve ter um antes de fazer sua armadilha (óbvio). Um navio pesqueiro só pode coletar em uma Armadilha para Peixes por vez. Cada Armadilha fornece uma quantidade limitada de alimentos antes de danificar-se e ter que ser reconstruída. Quando uma Armadilha para Peixes fica danificada, o Navio Pesqueiro designado a ela se move para a Armadilha para peixes e se torna ocioso. Para reconstruir uma Armadilha para Peixes danificada, selecione o Navio pesqueiro ocioso e clique com o botão direito em Armadilha para Peixes.
