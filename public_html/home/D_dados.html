Unidades e ataques do AOK_TC


Apresentaremos aqui tabelas dos ataques, bônus de ataques e muito mais, isto é de fundamental importância para sabermos a unidade que devemos usas para um contra-ataque eficiente. As siglas usadas na tabela estão especificadas em logo a baixo:
 

LOS Numero de quadrados que a unidade consegue ver. Cada quadrado é = a um segmento de parede
MR Taxa de movimento: quadrados que uma unidade pode mover por segundo. Uma unidade com um MR de 1.0 viajará um quadrado por segundo enquanto uma unidade com um MR de 1.5 será mais rapidamente 50%!
FR Taxa de Ataque: Quanto tempo a unidade demora para realizar o ataque seguinte. Quanto menor este número mais rápido são os ataques das unidades. Um FR de 2.0 é o padrão. Skirmisher possui um FR de 3.0 s, isto significa dizer que a cada 3 seg ele ataca uma vez.
BT Tempo de Construção: Tempo para treinar uma unidade.
Custo Custo das unidades (F)food, (G)gold, (W)wood, and (S)stone.
HP Hit Points: Vida da unidade
AT ATack: Ataque da unidade.
PT Ataque penetrante: Quanto a flexa provoca de danos ao atacar.
AR Armadura: Quantia de dano que sofre uma unidade quando esta é atacada.
PA Armadura penetrante: quantia de dano que sofre uma unidade quando esta é atacada com flechas de é atacada
RA Gama: distância que uma unidade pode atacar.
NOTA: UM RA de 0 é para todas as tropas de combate corpo a corpo. Isto é DIFERENTE de LOS que é a visão.
